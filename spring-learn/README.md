# spring-learn

A minimal Spring Web Maven project created for Hands on 1.

How to build and run:

```bash
cd /home/jobin/Placement/cognizant/cognizant_deepskill/spring-learn
mvn clean package
mvn spring-boot:run
```

What to check:
- `src/main/java` — application code
- `src/main/resources` — configuration (application.properties)
- `src/test/java` — test sources
- `SpringLearnApplication.java` — main() and `@SpringBootApplication` annotation
- `pom.xml` — Maven configuration and dependencies
