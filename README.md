# Cognizant Digital Nurture 5.0 - Deep Skilling

This repository contains my hands-on coursework for the Cognizant Deep Skilling program, mapping my journey through Full Stack development. It encompasses foundational version control exercises with Git, backend architectural fundamentals using Java, Maven, and Spring Boot, as well as a comprehensive, fully functional "Student Course Portal" built with Angular 19. Throughout these projects, I've implemented modern paradigms covering robust routing, component-driven UI, mock REST API integrations (via Http Client and Interceptors), and enterprise state management architectures utilizing NgRx.
